github日常使用(二)版本库回退或返回

  • github
记点博客

  • 通过命令git status查看当前仓库的状态:

  • 通过命令git diff 查看文件曾被修改了什么内容:

    然后可以通过命令git status,git commit对文件进行操作

  • 通过命令git log查看版本历史记录,即是日志(显示从最近到最远的提交日志)

    输入后输出几列一大串数字是commit添加到仓库的id号.

  • 回退到先前的版本仓库

    1.查看当前版本,head表示当前版本,head^表示上一个版本,head^^表示上上个版本…再往前可以用head~10(第10个),这样加相应的版本数字.现在通过以下命令返回相应版本:

    2.通过命令cat查看文件内容是否为当前版本内容是否为返回的版本内容

  • 回退后再返回最新的版本

    通过命令git reflog找回之前每一次对版本文件的操作记录,这样就可以查找到最新的版本的ID号

    知道ID号,就可以通过命令git reset ID号返回最新的版本了.
